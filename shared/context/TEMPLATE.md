# Executive Dashboard Template

Auto-generated HTML dashboard template for executive reporting via MCP dispatch.

## Usage with report-generator

```python
from exec_template import generate_dashboard_html, KPICard, Section, Insight

# Define KPIs
kpis = [
    KPICard("Ticket Médio", "R$501", accent=True),
    KPICard("PA", "1,5"),
    KPICard("Venda Bruta", "R$18,2M"),
    KPICard("Dev %", "2,3%"),
]

# Define sections (can contain tables, charts rendered as HTML, etc.)
sections = [
    Section("Produtos Top 5", "<table>...</table>"),
    Section("Tendência Semanal", "<svg>...</svg>"),
]

# Define insights/alerts
insights = [
    Insight("Crescimento de 15% vs semana anterior", type="positive"),
    Insight("Taxa de devolução acima do esperado", type="alert"),
]

# Generate HTML
html = generate_dashboard_html(
    title="Análise de Produto",
    brand="FARM",
    period="11–17 abr 2026",
    hero_label="Venda Líquida Total",
    hero_value="R$12,0M",
    hero_sub="35.778 peças · 23.977 transações · 3.394 produtos ativos",
    kpis=kpis,
    sections=sections,
    insights=insights,
)

# Output: complete HTML file, ready to publish to Vercel
print(html)
```

## API Reference

### `generate_dashboard_html()`

Generate a complete standalone HTML dashboard.

**Parameters:**
- `title` (str): Dashboard title. E.g., "Análise de Produto"
- `brand` (str): Brand/company name. E.g., "FARM", "Soma"
- `period` (str): Date period. E.g., "11–17 abr 2026"
- `hero_label` (str): Main KPI label. E.g., "Venda Líquida Total"
- `hero_value` (str): Main KPI value. E.g., "R$12,0M"
- `hero_sub` (str, optional): Context below hero. E.g., "35.778 peças · ..."
- `kpis` (list[KPICard], optional): KPI cards for the KPI row (2–4 recommended)
- `sections` (list[Section], optional): Content sections (can include tables, charts as HTML)
- `insights` (list[Insight], optional): Insights/alerts to display

**Returns:**
- Complete HTML string (single file, inlined CSS, no external dependencies)

### `KPICard`

Dataclass for a KPI card in the hero row.

```python
@dataclass
class KPICard:
    label: str           # E.g., "Ticket Médio"
    value: str           # E.g., "R$501"
    accent: bool = False # If True, uses blue-soft instead of blue-light
```

### `Section`

Dataclass for a content section.

```python
@dataclass
class Section:
    title: str      # E.g., "Produtos Top 5"
    content: str    # Raw HTML (table, chart SVG, paragraphs, etc.)
```

### `Insight`

Dataclass for an insight/alert.

```python
@dataclass
class Insight:
    text: str                           # The insight text
    type: str = "neutral"  # One of: neutral, positive, alert, highlight
```

## Integration with MCP

The template generator is used by `publicar_dashboard()` in the MCP server to render dashboards generated by Claude when executives ask for analyses. Claude generates the `html_content` (which can use this template), and the MCP publishes it to Vercel.

## Coloring Guide

- **Neutral insights**: Gray border (default)
- **Positive insights** (growth, improvement): Blue-soft border
- **Alerts** (issues, anomalies): White border
- **Highlights** (interesting patterns): Blue-light border

KPI values use blue-light by default; use `accent=True` for blue-soft.
